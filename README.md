````markdown
## 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/AhsanLaeeq/Ecommerce-store.git
cd Ecommerce-store
````

Install dependencies:

```bash
npm install
```

Run locally:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Deploy with GitHub Pages:

```bash
npm run deploy
```
